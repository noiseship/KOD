# 💰 Expense Tracker

**Автор:** Вавилин Матвей

---

## 📝 Описание

Expense Tracker - это десктопное приложение для отслеживания личных расходов. 
Позволяет добавлять расходы, фильтровать по категориям и датам, 
просматривать статистику и сохранять данные в JSON.

---

## 🚀 Установка и запуск

### Требования
- Python 3.7 или выше
- Tkinter (входит в стандартную установку Python)

### Запуск

```bash
# Скачайте репозиторий
git clone https://github.com/your-username/ExpenseTracker.git
cd ExpenseTracker

# Запустите приложение
python expense_tracker.py

# Запуск тестов
python tests.py
